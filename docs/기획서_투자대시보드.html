AI 기반 멀티팩터 투자 분석 대시보드 서비스 기획서

문서 버전 v1.0 작성일 2026-04-15 작성 부서 Product Management 문서 분류 서비스 기획서 (Service Requirements Document)

Executive Summary

Stock Manager는 40개 정량 팩터와 규칙 기반 분석 엔진을 결합하여, 투자 데이터의 수집·분석·시각화·인사이트 생성을 자동화하는 범용 금융 분석 대시보드입니다.

개인 투자자가 겪는 세 가지 구조적 문제 — ① 분석 기준의 주관성, ② 데이터 품질의 불균일성, ③ 해석 결과의 재현 불가능성 — 을 Skills.md라는 단일 규칙 문서 체계로 해결하며, 티커·테마·포트폴리오 등 다양한 입력을 동일한 분석 파이프라인에서 처리합니다.

본 기획서는 서비스의 가치 제안, 분석 아키텍처, 대시보드 구성, Skills.md 설계 방향, 그리고 확장 로드맵을 정의합니다.

1. 서비스 개요

1.1 서비스 정의

Stock Manager는 Skills.md에 정의된 일관된 분석 규칙을 기반으로, 투자 데이터를 자동으로 수집·분석·시각화하고 실행 가능한 투자 인사이트를 생성하는 AI 투자 분석 대시보드 서비스입니다.

1.2 문제 정의 (Problem Statement)

개인 투자자 및 실무 애널리스트는 다음과 같은 반복적 문제에 직면합니다.

영역 기존 문제 Stock Manager 해결 방식
데이터 수집 데이터 구조가 매번 달라 일관된 분석이 어려움 티커·테마·포트폴리오 입력을 단일 파이프라인으로 자동 처리하며, 8단계 품질 검증을 수행
분석 기준 분석 기준이 주관적이고 편향됨 40개 정량 팩터와 섹터별 동적 가중치를 통해 객관적 스코어링 체계 구축
시각화 차트 선택에 전문 지식이 필요 데이터 유형에 따라 차트 자동 매핑 규칙 적용
해석 지표 해석이 어렵고 추상적 6단계 투자 시그널과 자연어 인사이트, LLM 기반 리서치 리포트 생성
재현성 동일 데이터에 대한 분석 결과가 일관되지 않음 Skills.md 규칙에 따라 누가 실행해도 동일한 결과 보장

1.3 핵심 차별점 (Differentiators)

1. 규칙 기반 범용성 (Rule-based Universality) Skills.md 하나로 모든 분석 로직이 명시적으로 정의됩니다. 새로운 데이터가 입력되어도 동일한 규칙으로 분석이 수행됩니다.

2. 멀티팩터 통합 스코어링 (Multi-factor Scoring) 재무(20) + 기술적(10) + 감성(10) 팩터를 통합한 0~100 복합 스코어를 제공합니다. 단일 지표가 아닌 다차원 관점으로 종목을 평가합니다.

3. 시장 적응형 가중치 (Adaptive Weighting) SPY 60일 수익률을 기반으로 강세장·약세장·중립 시장을 판정하고, 팩터 가중치에 보정 승수를 동적으로 적용합니다. 11개 섹터별 가중치 테이블도 별도로 관리됩니다.

4. 실행 가능한 인사이트 (Actionable Insight) “Strong Buy ~ Strong Sell” 6단계 시그널, 포트폴리오 4단계 건강 진단, 딥 바잉(역발상 매수) 기회 탐지 등 즉시 실행 가능한 판단 근거를 제공합니다.

5. LLM 기반 리서치 리포트 (AI Research Report) Google Gemini 3.1 Flash-Lite를 활용하여, 수집된 정량 데이터와 관련 뉴스를 종합한 한국어 투자 리서치 리포트를 각 페이지 하단에서 원클릭 생성합니다.

1.4 타겟 사용자 (Target Users)

사용자 유형 활용 시나리오 기대 효과
개인 투자자 보유 종목 건강 진단, 신규 투자 후보 탐색 객관적 판단 근거 확보
투자 커뮤니티 테마별 종목 분석 공유, 랭킹 비교 토론 품질 제고
금융 애널리스트 멀티팩터 기반 리포트 자동 생성 리서치 시간 단축
핀테크 서비스 API 연동을 통한 투자 분석 기능 탑재 서비스 차별화

1.5 비즈니스 가치 (Business Value)

  • 시간 절감: 40개 팩터 수집·계산 프로세스 자동화로 종목 분석 소요 시간을 90% 이상 단축
  • 분석 품질 표준화: Skills.md 규칙 기반으로 분석 결과의 일관성과 재현성 확보
  • 확장성: 섹터·시장·입력 형식 등 주요 축에 대한 범용 설계로 신규 도메인 추가 용이
  • AI 네이티브: LLM 리포트 기능으로 정량 데이터와 정성적 해석을 하나의 사용자 경험으로 통합

2. 분석 흐름 설계

2.1 전체 분석 아키텍처

Stock Manager의 분석 엔진은 다음 4개 레이어로 구성됩니다.

① 입력 레이어 사용자는 세 가지 방식으로 분석을 시작합니다 — 티커 직접 입력(예: AAPL, MSFT), 테마 키워드 선택(예: AI, 반도체), 포트폴리오 입력(TICKER:수량@진입가 형식).

② 데이터 수집 레이어 - 시장 데이터: yfinance, Finnhub - 뉴스 및 감성: Google News RSS, VADER 감성 분석기 - 공시 정보: SEC EDGAR (8-K, 10-Q, 10-K)

③ 분석 엔진 레이어 (Skills.md 기반) - 데이터 품질 검증 (8개 자동 체크) - 재무 분석 (20개 지표) + 기술적 분석 (10개 지표) + 감성 분석 (10개 지표) - Z-Score 정규화 + 섹터별 동적 가중치 + 시장 상황 보정 승수 - 복합 스코어 (0~100) + 딥 보너스 가산

④ 출력 레이어 - 투자 시그널 6단계 판정 (Strong Buy ~ Strong Sell) - 차트 자동 매핑 및 렌더링 (Recharts 기반) - 자연어 인사이트 카드 - LLM 종합 분석 리포트 (Gemini 3.1 Flash-Lite)

프론트엔드는 Next.js 16(App Router) + Tailwind CSS v4로 구현되며, 백엔드 FastAPI 서버와 REST API로 통신합니다.

2.2 분석 파이프라인 상세

Stock Manager는 사용자 시나리오에 따라 네 가지 분석 파이프라인을 제공합니다.

Pipeline A — 테마 분석 (Theme Analysis)

AI가 추천하는 트렌딩 테마 또는 사용자가 직접 입력한 테마 키워드 하나로 관련 종목 발굴부터 랭킹까지 자동 수행합니다.

  1. 테마 → 관련 ETF 매핑 (예: AI → BOTZ, AIQ)
  2. ETF Top Holdings 추출로 후보 종목 리스트 생성
  3. 시가총액 필터링 (AI 테마 기준 $10B 이상)
  4. 3개 검색 쿼리로 뉴스 수집 (산업 동향 / 수요 전망 / 규제 이슈)
  5. 종목별 SEC 공시 3건 수집 (8-K, 10-Q, 10-K)
  6. 40개 팩터 수집 및 스코어링
  7. 딥 보너스 계산 (낙폭 + 반등 확인)
  8. 종합 랭킹 생성

출력: 테마 인사이트 카드 + 점수 바 차트 + 상세 랭킹 테이블 + LLM 종합 리포트

Pipeline B — 포트폴리오 진단 (Portfolio Diagnosis)

보유 종목, 수량, 진입가를 입력하면 실시간 P&L 추적과 건강 상태 진단을 수행합니다.

  1. 종목별 현재가 조회 및 P&L 계산
  2. 4단계 Phase 분류 (Uptrend / Stable / Unstable / Critical)
  3. 포트폴리오 건강도 점수 산출 (0~100)
  4. 리스크 알림 및 긍정 시그널 분류
  5. 포트폴리오 구성 비율 및 종목별 P&L 시각화

출력: 요약 카드 4종 + 보유 종목 테이블 + 도넛 차트 + P&L 바 차트 + 알림 카드 + LLM 종합 리포트

Pipeline C — 종합 종목 분석 (Comprehensive Analysis)

단일 종목에 대한 딥다이브 분석을 수행합니다.

  1. 재무 20개 지표 수집 및 분석
  2. 기술적 10개 지표 계산 및 시그널 판정
  3. 감성 10개 지표 분석 (뉴스 + 애널리스트 컨센서스 + 시장 심리)
  4. 복합 스코어 산출 (섹터 가중치 + 시장 보정)
  5. 투자 판단 근거 및 리스크 평가 생성
  6. 팩터별 해석 (재무 / 기술적 / 감성 각각의 자연어 인사이트)

출력: 투자 시그널 카드 + 6대 팩터 레이더 차트 + 투자 근거 + 리스크 + 펀더멘털 요약 + 가격 차트 + LLM 종합 리포트

Pipeline D — 딥 바잉 레이더 (Dip Buying Radar)

낙폭 과대 + 반등 조짐 종목을 자동 탐지합니다.

  1. 180일 고점 대비 낙폭 계산
  2. 10일 반등 모멘텀 확인
  3. 딥 보너스 산식 적용: Dip Bonus = 0.5 × 낙폭점수 + 0.5 × (낙폭 × 모멘텀)
  4. 복합 점수와 합산하여 최종 스코어링

출력: 딥 바잉 후보 리스트 (낙폭률, 반등률, 시그널)

2.3 데이터 품질 보장

모든 분석에 앞서 8개 항목에 대한 자동 검증을 수행합니다.

검증 항목: ① 필수 컬럼(OHLC) 존재, ② 데이터 타입(숫자형), ③ 날짜 연속성(3일 초과 갭 없음), ④ 결측치 비율 1% 미만, ⑤ 이상치(3σ) 5건 미만, ⑥ 제로값 부재, ⑦ 가격 일관성(High ≥ Low), ⑧ 거래량 유효성.

품질 점수 체계

점수 등급 처리 방식
95% 이상 Excellent 정상 분석
85 ~ 95% Good 정상 분석
70 ~ 85% Fair 경고 표시 후 분석
50 ~ 70% Poor 자동 클리닝 후 분석
50% 미만 Critical 분석 불가 안내

3. 대시보드 구성

3.1 레이아웃 및 페이지 구조

서비스는 좌측 240px 사이드바와 우측 메인 콘텐츠 영역의 2단 구조를 채택합니다. 각 분석 페이지의 하단에는 LLM 종합 분석 리포트 섹션이 공통적으로 포함됩니다.

순번 페이지명 경로 역할
1 Market Overview / 시장 지수 및 섹터 성과 요약
2 Theme Explorer /theme AI 추천 테마별 종목 발굴 및 분석
3 Stock Analyzer /stock 개별 종목 40팩터 딥다이브
4 Portfolio Dashboard /portfolio 보유 종목 건강 진단 및 P&L 추적
5 Ranking Engine /ranking 프리셋 그룹 종목 비교 및 랭킹

별도의 리포트 센터 페이지 없이, 각 분석 페이지(2~5) 하단에 LLM 종합 분석 리포트 기능을 통합하여 데이터와 해석의 맥락이 분리되지 않도록 설계했습니다.

3.2 각 페이지 상세 구성

Page 1. Market Overview

시장의 전체적인 분위기를 한눈에 파악하는 진입 페이지로, 사용자 접속 시 자동으로 데이터가 로딩됩니다.

  • 시장 상황 게이지: Bull / Neutral / Bear 판정과 SPY 60일 수익률 표시
  • 주요 지수 카드 (4종): S&P 500, NASDAQ, DOW, VIX — 각각 현재가, 등락률, 30일 미니 AreaChart
  • 섹터별 성과 바 차트: 11개 섹터(Technology, Financial, Healthcare, Energy, Consumer, Industrials, Utilities, Real Estate, Materials, Communication, Staples)의 1개월 수익률

Page 2. Theme Explorer

AI 추천 트렌딩 테마 또는 사용자 입력 테마로 관련 종목을 자동 발굴하고 분석합니다.

  • 입력 방식: AI 추천 트렌딩 테마 pill 버튼 + 직접 입력 텍스트 필드
  • 테마 인사이트 카드: 테마의 최상위 종목과 점수 기반 요약 문장
  • Top Stocks 점수 바 차트: 종합 점수 기준 BarChart (1위 파란색, 2~3위 청록색, 나머지 회색 그라데이션)
  • 종목 랭킹 테이블: 순위, 종목명, 종합 점수, 시그널 배지, 재무/기술/퀄리티 점수, 섹터
  • LLM 종합 분석 리포트: 하단 ‘리포트 생성’ 버튼

Page 3. Stock Analyzer

단일 종목에 대한 40팩터 심층 분석 대시보드입니다.

  • 입력 방식: 티커 검색 바 (예: NVDA 입력 후 분석)
  • 투자 시그널 카드: 복합 점수(0~100), Buy/Hold/Sell 배지, 투자 판단 이유
  • 6대 팩터 레이더 차트: Financial, Technical, Sentiment, Growth, Quality, Value
  • 투자 판단 근거 카드: Decision + Confidence + 매수 근거 리스트
  • 리스크 평가 카드: 위험 요소 리스트 또는 “특별한 리스크 없음”
  • 팩터별 해석 카드 (3종): 재무 / 기술적 / 감성 분석 각각의 자연어 인사이트
  • 핵심 펀더멘털 요약: Market Cap, P/E, EPS, P/B, ROE, Profit Margin, Revenue Growth, Sector
  • 6개월 가격 차트: 그라데이션 AreaChart
  • LLM 종합 분석 리포트: 하단 ‘리포트 생성’ 버튼

Page 4. Portfolio Dashboard

보유 종목의 건강 상태, P&L, 리스크 알림을 종합 모니터링합니다.

  • 입력 방식: TICKER:수량@진입가 형식 (예: AAPL:50@150, MSFT:30@350) + 현금 입력
  • 요약 카드 (4종): 총 자산, 총 손익, 건강도 점수, Phase
  • 보유 종목 분석 테이블: 종목, 수량, 진입가, 현재가, P&L 금액, P&L %, 점수, 시그널 배지
  • 포트폴리오 구성 도넛 차트: 종목별 비중 + 현금 비중
  • 종목별 P&L 수평 바 차트: 이익(녹색) / 손실(빨간색)
  • 리스크 알림 카드: 변동성, 데드 크로스 등 경고
  • 긍정 시그널 카드: 상승 추세, 수익성 양호 등
  • LLM 종합 분석 리포트: 하단 ‘리포트 생성’ 버튼

Page 5. Ranking Engine

4. Skills.md 설계 방향

4.1 설계 철학

Skills.md는 분석 규칙의 “단일 진실 원천(Single Source of Truth)”으로 설계되었습니다. 모든 분석 로직은 이 문서에 숫자와 수식으로 명시되며, 구현은 문서의 규칙을 코드로 변환하는 방식을 취합니다.

핵심 원칙 4가지

  1. 규칙 명시성: 모든 임계값, 가중치, 수식을 숫자로 명시합니다. 예: “RSI > 70 → 과매수”는 허용하되, “RSI가 높으면 과매수”는 배제합니다.
  2. 재현 가능성: 동일한 데이터와 동일한 Skills.md 규칙이 적용되면 항상 동일한 결과가 산출되어야 합니다.
  3. 확장 가능성: 새로운 팩터나 섹터를 추가할 때 기존 구조를 변경하지 않고 규칙만 추가하면 되도록 설계합니다.
  4. 범용성: 특정 종목이나 시장에 종속되지 않는 일반화된 규칙을 지향합니다. 미국·한국 등 다중 시장 대응이 가능해야 합니다.

4.2 Skills.md 구성 체계

Skills.md는 다음 8개 섹션으로 구성됩니다.

섹션 내용
1. 데이터 분석 기준 정의 6대 팩터 체계, 11개 섹터별 동적 가중치, 3단계 시장 상황 보정 규칙, Z-Score 정규화 4단계
2. 투자 지표 계산 규칙 재무 팩터 20개, 기술적 지표 10개, 감성 팩터 10개, 복합 스코어, 딥 보너스, 포트폴리오 4단계 진단, 모멘텀, 백테스트 검증
3. 시각화 선택 기준 차트 자동 매핑, 색상 팔레트, 반응형 레이아웃
4. 리포트 구성 흐름 테마 분석 파이프라인(8 Step), 포트폴리오 진단, 종합 분석, 딥 후보 탐색
5. 인사이트 생성 규칙 9점 만점 종합 판정, 감성 모멘텀 판정(5단계), 감성 신뢰도, 트렌딩 감지, 자연어 템플릿(6종), 위험 경고 조건
6. 데이터 품질 관리 8개 검증 체크, 5단계 품질 점수, 3종 자동 클리닝
7. 범용성 설계 다중 시장 지원(US/KR), 3가지 입력 형식(티커/테마/CSV), 6단계 TTL 캐시 정책
8. 대시보드 페이지 구성 5개 페이지 구성 요소 + LLM 종합 리포트

4.3 바이브코딩 활용 전략

Skills.md는 단순 문서가 아니라 AI 개발 에이전트(Claude)에게 전달하는 실행 가능한 스펙입니다.

  • “Skills.md 섹션 2.2의 RSI 계산 규칙을 기반으로 technical_indicators.py를 구현”
  • “섹션 3.1의 차트 매핑 규칙대로 데이터 유형별 차트 컴포넌트를 자동 선택하는 ChartSelector 컴포넌트 생성”
  • “섹션 5.5의 자연어 템플릿을 사용하여 투자 시그널을 인사이트 카드로 렌더링”
  • “섹션 4.1 테마 분석 파이프라인의 8단계 흐름을 API로 구현”

이러한 접근은 문서가 곧 스펙이고, 스펙이 곧 코드로 변환되는 개발 흐름을 만들어, 분석 로직의 변경이 항상 Skills.md에서 시작되도록 보장합니다.

8. 결론

Stock Manager는 Skills.md에 정의된 40개 팩터 분석 규칙을 기반으로, 투자 데이터를 자동으로 수집·분석·시각화하고 실행 가능한 인사이트를 생성하는 범용 금융 투자 대시보드입니다.

핵심 경쟁력 다섯 가지

  1. Skills.md — 분석의 단일 진실 원천: 40개 팩터, 차트 자동 매핑, 6단계 시그널이 모두 규칙으로 정의되어 재현 가능성과 확장성을 동시에 확보합니다.
  2. 바이브코딩 — 문서가 곧 코드: Skills.md 규칙을 AI가 코드로 변환하는 구조로, 로직 변경이 항상 문서에서 시작됩니다.
  3. 범용성 — 어떤 데이터든 분석 가능: 티커·테마·포트폴리오의 3가지 입력 형식, 미국·한국 다중 시장, 11개 섹터를 단일 파이프라인으로 처리합니다.
  4. 실행 가능한 인사이트 — 판단이 아닌 행동 지침: “Strong Buy” 시그널과 자연어 설명, 위험 경고를 통해 사용자가 즉시 행동할 수 있는 판단 근거를 제공합니다.
  5. LLM 종합 리포트 — AI 기반 리서치 자동화: Gemini 3.1 Flash-Lite 기반 한국어 리포트, 뉴스 기사 인용 분석, 각 페이지 하단 원클릭 생성으로 정량 분석과 정성 해석을 통합합니다.

Stock Manager는 단순한 분석 도구가 아닌, 규칙 기반 투자 분석의 표준 플랫폼을 지향합니다.

본 기획서는 Stock Manager v1.0 기준으로 작성되었으며, 제품 발전에 따라 업데이트될 수 있습니다.
